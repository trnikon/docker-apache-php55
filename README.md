Ubuntu + apache + php5.5 as apache module

Usage
------

```
$ docker run -d -P trnikon/docker-apache-php55
```

-------------------
* Ubuntu Server 14.04, based on ubuntu docker image
* apache2
* php5
* php5-cli
* libapache2-mod-php5
* php5-gd
* php5-mcrypt
* php5-zip
* php5-xml
* php5-iconv
